What this blog is going to be about

Aug 29, 2024 · 190 words · 1 minute read

I’ve always wanted to set up a vanity blog to showcase anything cool and awesome that I’ve worked on. Having seen a couple of my friends write blogs on WordPress and Blogger, these two blogging sites were the first to pop into my mind. However, for a simple use-case like mine, it seemed complicated and unnecessary. All I needed was a domain, a place to host my content, and an HTML generator to turn simple text into beautiful HTML.

I came across the combination of GitHub Pages and Jekyll. This seemed like the perfect setup for me. However, I ran into some errors when installing Jekyll, and I did not have the patience to debug it. I started searching for an alternative and discovered Hugo.

Hugo is a static website generator written in Go. You can quickly build static websites such as this one, with your choice of theme. I used the theme mini to build this website. All you have to do is, install Hugo and write your content in Markdown format. Hugo takes care of the rest. You can check out the quick start guide for more details.
